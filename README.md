# pokedex
# 🧭 Pokédex Web App

## 📌 Overview

This project is an interactive Pokédex web application that allows users to search for Pokémon and view detailed information in real time. It uses data from an external API to dynamically display content on the page.

---

## 🚀 Features

* 🔍 Search for Pokémon by name
* 📊 Display key information:

  * Name
  * Image
  * Type
  * Stats
* 🌐 Fetch live data from an external API
* ⚡ Dynamic page updates using JavaScript
* 📱 Simple and responsive interface

---

## 🛠️ Technologies Used

* **HTML** – structure of the application
* **CSS** – styling and layout
* **JavaScript** – functionality and logic
* **Fetch API** – for making API requests
* **PokéAPI** – data source
  https://pokeapi.co

---

## ⚙️ How It Works

The application sends a request to the PokéAPI using the `fetch()` function.
When a user searches for a Pokémon, the app retrieves the data asynchronously using `async/await`, then updates the webpage with the results.

This demonstrates how modern web apps communicate with APIs and render dynamic content.
