ℹ️ About

Welcome to the Pokédex — your ultimate companion for exploring Pokémon data! This is an interactive web application that lets you search, compare, and save your favorite Pokémon with ease.

Features

  • Search - Find any Pokémon by name or ID
  • View Stats - See HP, Attack, Defense, Height, Weight, and Types
  • Compare - Side-by-side comparison of two Pokémon
  • Favorites - Save your favorite Pokémon with IndexedDB storage
  • Export - Download your favorites as a JSON file
  • Random - Discover random Pokémon
  • Search History - Quick access to recently viewed Pokémon
  • Dynamic Background - Background changes based on Pokémon type
  • Sound Effects - Audio feedback when searching
  • Pokémon Details - View full stats, abilities, and evolution chains
  • Responsive Design - Works great on mobile and desktop

Data Source

This app is powered by the PokéAPI, an extensive and well-maintained API that provides comprehensive Pokémon data including sprites, stats, types, and more.

Technology Stack

  • HTML5 - Modern semantic markup
  • CSS3 - Responsive design with gradients and animations
  • Vanilla JavaScript - No frameworks, pure client-side logic
  • IndexedDB - Browser storage for favorites and offline support
  • Web Audio API - Sound effects
  • Fetch API - Real-time data fetching
